학생을 위한 학습 안내
각 모듈의 사용법을 아래에서 확인하세요. 섹션을 눌러 펼칠 수 있습니다.
🚀 시작하기
▼
  1. 앱을 열면 홈 화면이 나타납니다. 상단에서 레슨(채널)을 선택하세요.
  2. 3가지 모듈 중 원하는 것을 선택: 단어 학습, 독해 연습, 번역 연습
  3. 학습을 마치면 진행률이 자동 저장됩니다.
📚 단어 학습
▼
  1. 단어 카드에서 단어, 발음, 뜻, 예문을 한눈에 확인합니다. 🔊 버튼으로 발음을 들어보세요.
  2. 이전/다음 버튼으로 자유롭게 단어를 탐색합니다.
  3. 모든 단어를 본 후 셔플 퀴즈가 시작됩니다. 빈칸에 맞는 단어를 고르세요.
  4. 틀린 단어가 있으면 오답만 다시 버튼으로 재학습할 수 있습니다.
📖 독해 연습
▼
  1. 지문 목록에서 학습할 지문을 선택합니다.
  2. 🔊 지문 듣기로 전체 지문을 들을 수 있습니다.
  3. 문장별 연습을 펼치면 문장 하나씩 듣고 받아쓰기를 할 수 있습니다.
  4. 지문 아래에 문제가 바로 표시됩니다. 스크롤해서 풀어보세요.
✏️ 번역 연습
▼
  1. 한국어 → 영어 또는 영어 → 한국어로 번역합니다.
  2. 막히면 힌트 버튼을 눌러 도움을 받으세요 (최대 3단계).
  3. 제출하면 핵심 표현 포함 여부가 시각적으로 표시됩니다.
⚙️ 속도 & 녹음 설정
▼

홈 화면 하단의 SPEED로 TTS 속도를 조절할 수 있습니다.

  • 느리게 (0.6배) — 처음 듣는 단어에 적합
  • 보통 (0.85배) — 기본 설정
  • 빠르게 (1.0배) — 실전 연습용

발음 녹음 토글을 켜면 단어 학습 시 본인 발음을 녹음하고 비교할 수 있습니다.

👩‍🏫
교사/관리자 가이드
콘텐츠를 만들고 관리하는 방법을 안내합니다.
🔑 관리자 페이지 접속
▼

홈 화면 하단의 ⚙️ 관리자 페이지를 누르면 이동합니다.
로그인 없이 바로 사용 가능합니다. (학생에게는 URL을 공유하지 않는 것을 권장)

🚀 원클릭 추출 (가장 쉬운 방법)
▼
  1. 관리자 페이지에서 🤖 자동 추출 탭을 선택합니다.
  2. 레슨 ID 입력 (예: lesson01, lesson02)
  3. 지문 제목을 입력합니다.
  4. 교과서의 영어 지문을 복사해서 붙여넣습니다.
  5. 한국어 해석을 붙여넣습니다. (문장 순서가 영어와 동일해야 합니다)
  6. 🚀 원클릭: 추출 + API조회 + 앱 저장 버튼을 누릅니다.
  7. 자동으로 단어 추출 → 발음/뜻 조회 → 독해 생성 → 번역 생성 → 앱 저장이 완료!
✏️ 개별 편집
▼

각 탭에서 세밀하게 편집할 수도 있습니다:

  • 📚 단어 탭 — 단어 추가/삭제, 🔍 자동 조회(발음/뜻/예문), CSV 가져오기
  • 📖 독해 탭 — 지문 추가, 객관식/주관식 문제 만들기
  • ✏️ 번역 탭 — 번역 문장 추가, 힌트/키워드 설정

편집 후 🚀 앱에 바로 저장을 누르면 즉시 반영됩니다.

자주 묻는 질문
앱을 설치해야 하나요?
▼

아니요. 웹 브라우저에서 바로 사용합니다. 원하시면 "홈 화면에 추가"로 앱처럼 설치할 수도 있습니다.

인터넷이 없어도 되나요?
▼

첫 접속 시 인터넷이 필요합니다. 이후 PWA 캐싱으로 일부 기능은 오프라인 사용 가능합니다. 단, TTS(음성)와 API 조회는 인터넷이 필요합니다.

다른 기기에서도 진도가 보이나요?
▼

아니요. 각 기기에 독립 저장됩니다. 다른 기기에서 사용하려면 JSON 내보내기/가져오기를 이용하세요.

소리가 안 나와요
▼

기기 볼륨을 확인하세요. 일부 브라우저는 화면을 한 번 터치해야 소리 재생이 가능합니다. 크롬 사용을 권장합니다.

데이터가 사라졌어요!
▼

브라우저 데이터(캐시, 쿠키 등)를 삭제하면 학습 데이터도 함께 사라집니다. 백업 파일이 있다면 관리자 페이지에서 복원할 수 있습니다.

새 레슨은 어떻게 학생에게 전달하나요?
▼

같은 기기에서는 새로고침하면 자동 반영됩니다. 다른 기기에는 JSON 파일을 공유하거나, GitHub에 데이터를 올려 배포하세요.
