ENG LIVE 도움말

📡 ENG LIVE란?

ENG LIVE는 고등학생을 위한 실시간 방송 스타일 영어 학습 앱입니다.
설치 없이 웹 브라우저에서 바로 사용할 수 있으며, 인터넷만 있으면 어디서든 학습할 수 있습니다.

  • 단어 학습 — 단어 카드 + 발음 듣기 + 퀴즈
  • 독해 연습 — 지문 읽기 + 문장별 듣기 연습 + 문제 풀기
  • 번역 연습 — 한→영 / 영→한 번역 + 힌트 + 자동 채점
🎓 학생 사용법

학생 시작하기

  1. 앱을 열면 홈 화면이 나타납니다. 상단에서 레슨(채널)을 선택하세요.
  2. 3가지 모듈 중 원하는 것을 선택하세요: 단어 학습, 독해 연습, 번역 연습
  3. 학습을 마치면 자동으로 진행률이 저장됩니다.

학생 단어 학습

  1. 단어 카드를 보며 단어·발음·뜻·예문을 확인합니다. 🔊 버튼으로 발음을 들어보세요.
  2. 이전/다음 버튼으로 단어를 자유롭게 탐색할 수 있습니다.
  3. 모든 단어를 본 후 퀴즈가 시작됩니다. 빈칸에 맞는 단어를 고르세요.
  4. 퀴즈 결과를 확인하고, 원하면 다시 도전하세요.

학생 독해 연습

  1. 지문 목록에서 학습할 지문을 선택합니다.
  2. 🔊 지문 듣기로 전체 지문을 들을 수 있습니다.
  3. 문장별 연습을 누르면 문장 하나씩 듣고 → 받아쓰기를 할 수 있습니다.
  4. 속도 조절: 느리게 / 보통 / 빠르게 중 내 수준에 맞는 속도를 선택하세요.
  5. 문제 풀기를 누르면 객관식·주관식 문제를 풀 수 있습니다.

학생 번역 연습

  1. 한국어 문장이 주어지면 영어로 번역하세요 (또는 그 반대).
  2. 막히면 힌트 버튼을 눌러 도움을 받으세요. 힌트는 3단계까지 있습니다.
  3. 제출하면 자동으로 채점됩니다. 핵심 키워드 포함 여부를 확인해줍니다.

학생 속도 설정

홈 화면 하단의 SPEED 버튼으로 TTS(음성) 속도를 조절할 수 있습니다.
느리게(0.6배) · 보통(0.85배) · 빠르게(1.0배) 중 선택하세요.
설정은 자동 저장되며, 독해 연습의 문장별 연습에서도 바로 변경할 수 있습니다.

👩‍🏫 교사·관리자 가이드

교사 관리자 페이지 접속

홈 화면 하단의 ⚙️ 관리자 페이지를 누르면 콘텐츠 관리 화면으로 이동합니다.
로그인 없이 바로 사용 가능합니다. (학생에게는 URL을 따로 공유하지 않는 것을 권장)

교사 가장 쉬운 방법: 원클릭 추출

  1. 관리자 페이지에서 🤖 자동 추출 탭을 선택합니다.
  2. 레슨 ID를 입력합니다 (예: lesson01, lesson02 등).
  3. 지문 제목을 입력합니다.
  4. 교과서의 영어 지문을 복사해서 붙여넣습니다.
  5. 한국어 해석을 복사해서 붙여넣습니다. (문장 순서가 영어와 동일해야 합니다)
  6. 🚀 원클릭: 추출 + API조회 + 앱 저장 버튼을 누릅니다.
  7. 자동으로 단어 추출 → 발음/뜻 조회 → 독해 지문 생성 → 번역 문장 생성 → 앱 저장이 완료됩니다.
  8. 학생 앱을 새로고침하면 새 레슨이 나타납니다.

교사 개별 편집

각 탭에서 세밀하게 편집할 수도 있습니다:

  • 📚 단어 탭 — 단어 추가/삭제, 🔍 자동 조회(발음·뜻·예문), CSV 가져오기
  • 📖 독해 탭 — 지문 추가, 객관식/주관식 문제 만들기
  • ✏️ 번역 탭 — 번역 문장 추가, 힌트·키워드 설정

편집 후 🚀 앱에 바로 저장 버튼을 누르면 즉시 반영됩니다.

교사 데이터 백업 (JSON)

💾 JSON 다운로드로 현재 데이터를 파일로 백업할 수 있습니다.
📂 JSON 불러오기로 백업한 파일을 다시 불러올 수 있습니다.
중요: 브라우저 데이터를 삭제하면 학습 데이터가 사라지므로, 주기적으로 JSON 백업을 권장합니다.

💾 데이터 저장 원리

이 앱은 서버나 데이터베이스를 사용하지 않습니다.
모든 데이터는 브라우저의 localStorage에 저장됩니다.

⚠️ 알아두세요

  • 같은 기기·같은 브라우저에서만 데이터가 유지됩니다.
  • 브라우저 캐시/데이터를 삭제하면 모든 학습 데이터가 사라집니다.
  • 다른 기기로 데이터를 옮기려면 JSON 내보내기 → 가져오기를 이용하세요.
  • 개인정보를 수집하지 않으며, 모든 데이터는 사용자 기기에만 존재합니다.

❓ 자주 묻는 질문
Q. 앱을 설치해야 하나요?
아니요. 웹 브라우저에서 바로 사용합니다. 크롬, 사파리, 엣지 등 모든 브라우저에서 동작합니다.
Q. 인터넷이 없어도 되나요?
첫 접속 시 인터넷이 필요합니다. 이후 데이터가 브라우저에 저장되므로 일부 기능은 오프라인에서도 사용 가능하지만, TTS(음성)와 API 조회는 인터넷이 필요합니다.
Q. 다른 기기에서도 내 진도가 보이나요?
아니요. 데이터는 각 기기의 브라우저에 독립적으로 저장됩니다. 다른 기기에서 사용하려면 JSON 내보내기/가져오기를 이용하세요.
Q. 소리가 안 나와요.
기기의 볼륨을 확인하세요. 일부 브라우저는 화면을 한 번 터치한 후에야 소리 재생이 가능합니다. 크롬 사용을 권장합니다.
Q. 새 레슨을 추가하면 학생에게 어떻게 알려지나요?
같은 기기에서는 앱을 새로고침하면 자동으로 나타납니다. 다른 기기의 학생에게는 JSON 파일을 공유하거나, 같은 네트워크의 서버를 이용해야 합니다.
Q. 데이터가 사라졌어요!
브라우저 데이터(캐시, 쿠키 등)를 삭제하면 학습 데이터도 함께 사라집니다. 주기적으로 JSON 백업을 해두세요. 백업 파일이 있다면 관리자 페이지에서 복원할 수 있습니다.
